# Sally Search

Die Recherche-App für Second-Hand-Händler und Sammler.

## Ziel

Sally Search unterstützt bei der Bewertung und Recherche von:

- Vinyl
- LEGO
- CDs/DVDs
- Schleich
- weiteren Sammlerartikeln

## Geplante Funktionen

- Recherche-Assistent
- Preisvergleich
- Suchverlauf
- Barcode-Scanner
- OCR-Erkennung
- Spracheingabe
- Matrix-Erkennung für Vinyl
- LEGO-Erkennung
